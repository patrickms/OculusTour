A virtual reality tour with speech synthesis

Content provided by Pearson TopTen guides through the Pearson API at developer.pearson.com

Instructions

This demo requires a modern web browser with WebGL support. Most mobile web browsers are not supported.

Browsers which do not support the web speech API will exhibit poor performance.

Standard web view

The Oculus Rift version of the tour requires the vr.js addin at https://github.com/benvanik/vr.js/tree/master

View with Oculus Rift

Navigation

Use the arrow keys (or your turn head if you are using the Oculus Rift) to look around.

Press enter, or press the Control key twice in quick successions, to move to the next scene in the tour
